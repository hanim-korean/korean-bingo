사대문(四大門)은 조선시대 한양(지금의 서울)을 둘러싼 성벽에 있던 4개의 큰 문이에요.
동·서·남·북 방향에 하나씩 있어요.

The Four Great Gates (사대문) were the four main gates of the walled city of Hanyang — the old name for Seoul — built during the Joseon Dynasty.

동쪽 EAST

흥인지문

동대문 (Dongdaemun)

Gate of Rising Benevolence

흥 = rise / 인 = benevolence (인자함) / 지 = of / 문 = gate

현재도 남아 있어요. 지금은 동대문 쇼핑 지구로 유명해요.
Still standing today — now famous for the Dongdaemun shopping district.

현존 / Still exists

남쪽 SOUTH

숭례문

남대문 (Namdaemun)

Gate of崇 Revered Propriety

숭 = revere (높이다) / 례 = propriety / 문 = gate

대한민국 국보 1호예요. 2008년 화재로 소실, 2013년 복원.
National Treasure No. 1. Damaged by arson in 2008, restored in 2013.

국보 1호 / National Treasure #1

서쪽 WEST

돈의문

서대문 (Seodaemun)

Gate of Gross Benevolence

돈 = strengthen (두텁게 하다) / 의 = righteousness / 문 = gate

1915년 일제강점기에 철거돼서 지금은 없어요. 터만 남아 있어요.
Demolished during Japanese colonial rule in 1915. Only the site remains.

철거됨 / Demolished

북쪽 NORTH

숙정문

북대문 (Bukdaemun)

Gate of Solemn Governance

숙 = solemn (엄숙하다) / 정 = governance / 문 = gate

산 위에 있어 실제로는 거의 쓰이지 않았어요. 지금도 북악산에 남아 있어요.
Built on a mountain, rarely used. Still stands on Bugaksan today.

현존 / Still exists

왜 서대문(돈의문)만 없나요?
일제강점기(1910–1945)에 일본이 도로를 넓히면서 철거했어요. 지금도 복원을 논의 중이에요.

Why is Seodaemun (Donuimun) the only one missing?
Japan demolished it during the colonial period to widen the road. There are ongoing discussions about restoring it.
