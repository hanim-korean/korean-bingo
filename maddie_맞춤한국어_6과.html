🔍 5과 ↔ 6과 비교 — Key Difference

⭐
오늘의 핵심 포인트! 5과에서는 ~이/가 좋아요를 배웠어요.
6과에서는 목적어 표지 ~을/를이 붙는 좋아해요 / 싫어해요를 배워요.
Today's key: we add an object marker ~을/를 and the verb changes!
✅ 5과 — 이미 배운 것
~이/가 좋아요
아이스크림이 좋아요 → I like ice cream
피자가 좋아요 → I like pizza
뭐가 좋아요? → What do you like?
🆕 6과 — 오늘 배울 것
~을/를 좋아해요 / 싫어해요
불고기를 좋아해요 → I like bulgogi
라면을 싫어해요 → I dislike ramyun
뭘 좋아해요? → What do you like?

📏
을 vs 를 규칙 — When to use 을 or 를?
앞 글자에 받침(final consonant)이 있으면 → 을 (예: 라면을, 김치를)
앞 글자에 받침이 없으면 → 를 (예: 불고기를, 피자를)
Tip: 받침 있음 → 을 / 받침 없음 → 를
