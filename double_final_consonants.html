겹받침 Double Final Consonants

서로 다른 두 개의 자음으로 이루어진 받침 — 어떻게 읽고, 언제 발음이 바뀔까요?

1. Basic Concept — 기본 개념

겹받침이 뭐예요?

'ㄺ'처럼 서로 다른 두 개의 자음으로 이루어진 받침이에요. (같은 자음 두 개가 만나면 '쌍받침' — 예: ㄲ, ㅆ)
A "double final consonant" (겹받침) is made of two different consonants stacked under a vowel — like 'ㄺ' (ㄹ + ㄱ) in 닭. Two of the same consonant (like ㄲ, ㅆ) is called "ssang-batchim" (쌍받침).

먼저: 7 대표음 (Representative Sounds)
한국어 받침은 아무리 다양해도 실제로는 7개 소리로만 발음돼요. 겹받침도 결국 이 7개 중 하나로 발음됩니다.
No matter how the consonant is written, every Korean batchim is pronounced as one of these 7 sounds.

예) 잎은 'ㅍ'이지만 → [입] (ㅂ 소리). 넋은 'ㄳ'이지만 → [넉] (ㄱ 소리). Even though '잎' is written with ㅍ, it's pronounced as [입] using ㅂ. The same logic applies to double consonants.

Memory trick: only 3 double batchim use the SECOND consonant for sound — ㄺ, ㄻ, ㄿ. All other 8 use the FIRST. Memorize these 3 and you're set!

2. Rule 1 — 첫째 받침이 발음되는 경우 (8개)

규칙 1 First consonant rule
아래 8개 겹받침은 어말(단어 끝)이나 자음 앞에서 왼쪽(첫째) 자음의 대표음으로 발음돼요.
These 8 double batchim use the FIRST (left) consonant for pronunciation when at the end of a word or before another consonant.
겹받침대표음예시발음의미 (English)
ㄳ[ㄱ]넋[넉]spirit/soul
ㄵ[ㄴ]앉다[안따]to sit
ㄶ[ㄴ]많다[만타]to be many
ㄼ[ㄹ]여덟[여덜]eight
ㄽ[ㄹ]외곬[외골]single-mindedly
ㄾ[ㄹ]핥다[할따]to lick
ㅀ[ㄹ]싫다[실타]to dislike
ㅄ[ㅂ]없다[업따]to not exist / not have
패턴 보기: ㄳ, ㄵ, ㄶ에서는 왼쪽이 ㄴ/ㄱ 계열이라서 그게 살아남고, ㄼ·ㄽ·ㄾ·ㅀ는 모두 왼쪽이 ㄹ이라서 ㄹ 소리, ㅄ는 왼쪽이 ㅂ이라서 ㅂ 소리.
Notice the pattern — the surviving sound matches the LEFT consonant: ㄴ/ㄱ for the first three, ㄹ for the four ㄹ-starting ones, ㅂ for ㅄ.
왜 [안따], [업따] 같이 된소리(ㄸ, ㅃ)가 될까?
받침이 ㄱ/ㄷ/ㅂ 소리로 끝나면 그 다음 ㄷ, ㅈ, ㄱ 등이 자동으로 된소리로 바뀌어요. 이건 '된소리되기'라는 별도 규칙이에요.
When the batchim ends in a ㄱ/ㄷ/ㅂ sound, the following ㄷ, ㅈ, ㄱ becomes "tense" (ㄸ, ㅉ, ㄲ). That's a separate rule called "된소리되기" (fortis/tensification).

3. Rule 2 — 둘째 받침이 발음되는 경우 (3개) ⚡

규칙 2 Second consonant rule — must memorize!
이 3개 겹받침만 어말이나 자음 앞에서 오른쪽(둘째) 자음의 대표음으로 발음돼요. 이 3개를 따로 외우는 게 가장 효율적이에요!
ONLY these 3 double batchim use the SECOND (right) consonant. They're the exceptions — memorize these and the rest follow Rule 1 by default.
겹받침대표음예시발음의미 (English)
ㄺ[ㄱ]읽다[익따]to read
ㄺ[ㄱ]닭[닥]chicken
ㄻ[ㅁ]젊다[점따]to be young
ㄻ[ㅁ]삶[삼]life
ㄿ[ㅂ]읊다[읍따]to recite (poem)
왜 이 3개만 다를까? 'ㄺ, ㄻ, ㄿ'은 첫째 자음이 모두 ㄹ이지만, 뒤에 오는 ㄱ, ㅁ, ㅍ이 더 강한 소리라서 그쪽이 살아남아요.
Why these three? The second consonant (ㄱ, ㅁ, ㅍ) is "stronger" and dominates over the ㄹ on the left. This is why they break the default pattern.
⚠️ 외우는 팁: "ㄹ" + 더 강한 자음 → 더 강한 자음이 이김.
ㄺ → ㄱ 살아남, ㄻ → ㅁ 살아남, ㄿ → ㅂ(ㅍ) 살아남.

4. Exceptions — 예외 케이스 ⚠️

예외 1: 'ㄺ' 동사 어간 + 'ㄱ' 앞에서 verb stem rule
'ㄺ' 받침은 보통 [ㄱ]으로 발음되지만, 용언(동사·형용사)의 어간 끝 'ㄺ'은 'ㄱ' 앞에서 [ㄹ]로 발음돼요.
Normally ㄺ → [ㄱ]. But when ㄺ is at the end of a verb/adjective stem AND followed by ㄱ, it's pronounced as [ㄹ] instead.
예시발음의미
맑다 (어말)[막따]to be clear ← 보통 규칙
맑게 (ㄱ 앞)[말께]clearly ← 예외!
읽다 (어말)[익따]to read ← 보통 규칙
읽고 (ㄱ 앞)[일꼬]read and... ← 예외!
예외 2: '밟-' 동사 special verb — must memorize
'ㄼ' 받침은 보통 [ㄹ]로 발음돼요 (예: 여덟 [여덜], 넓다 [널따], 짧다 [짤따]). 그런데 동사 '밟-' (to step on) 한 단어만은 자음 앞에서 [밥]으로, 모음 앞에서는 정상 연음으로 [발ㅂ]으로 발음해요.
Normally ㄼ → [ㄹ]. But the verb stem '밟-' is the lone exception: it sounds like [밥] before consonants but reverts to [발ㅂ] before vowels (normal linking).
예시표준 발음설명
밟다[밥ː따]'밟-' + ㄷ → 자음 앞 [밥], ㄷ이 된소리 [따]
밟고[밥꼬]'밟-' + ㄱ → 자음 앞 [밥], ㄱ이 된소리 [꼬]
밟지[밥찌]'밟-' + ㅈ → 자음 앞 [밥], ㅈ이 된소리 [찌]
밟는[밤ː는]'밟-' + ㄴ → [밥는]에서 비음화 → [밤는]
밟아[발바]모음 ㅏ 앞에서는 정상 연음 → ㄹ 남고, ㅂ이 다음 음절로
여덟 (비교)[여덜]일반 ㄼ 단어 → [ㄹ] (정상 규칙)
넓다 (비교)[널따]일반 ㄼ 단어 → [ㄹ] (정상 규칙)
💡 3가지 핵심:
① 자음 앞 → [밥] + 뒤따르는 자음 된소리화 (밟다 [밥따], 밟고 [밥꼬], 밟지 [밥찌])
② ㄴ 앞 → [밥]에서 비음화 일어나 [밤]으로 (밟는 [밤는])
③ 모음 앞 → 정상 연음 [발ㅂ]으로 (밟아 [발바], 밟아요 [발바요])
장음 표시 'ː' 는 무엇인가요? '밟다 [밥ː따]'에서 'ː'는 모음을 길게 발음하라는 표시예요. 표준국어대사전에 등재된 발음이지만, 실제 회화에서는 한국 사람들도 거의 구분 없이 짧게 발음해요. 학습할 때는 '밥' 위치만 정확히 기억하면 충분해요.
The 'ː' symbol indicates a long vowel (in 밟다 [밥ː따]). It's in the official dictionary, but in everyday speech most Koreans don't distinguish vowel length anymore. As a learner, focus on getting [밥] right — the length is optional.
참고: 'ㄼ' 받침의 다른 단어들 (짧다 [짤따], 넓다 [널따], 얇다 [얄따], 떫다 [떨따])은 모두 정상적으로 [ㄹ]로 발음돼요. '밟-' 한 글자만 예외라는 점, 꼭 기억해 주세요!
All other ㄼ words (짧다, 넓다, 얇다, 떫다) follow the normal [ㄹ] rule. The verb stem '밟-' is the ONE odd one out.

5. Linking 연음 — 모음 앞에서

규칙 3: 겹받침 + 모음 시작 조사/어미 linking rule
겹받침 뒤에 모음으로 시작하는 조사·어미·접미사가 오면, 둘째 받침(오른쪽)이 다음 음절의 첫소리로 넘어가요. 첫째 받침은 그 자리에 그대로 남아요.
When a vowel-starting particle/ending follows a double batchim, the SECOND (right) consonant moves over to start the next syllable. The first stays where it is.
값 + 이 → 갑 + 시
값이 → [갑씨] price + topic marker · 'ㅅ' becomes [ㅆ]
넋 + 이 → 넉 + 시
넋이 → [넉씨]
읽 + 어요 → 일 + 거요
읽어요 → [일거요] to read
앉 + 아요 → 안 + 자요
앉아요 → [안자요] to sit
없 + 어요 → 업 + 써요
없어요 → [업써요] 'ㅅ' → [ㅆ]
싫 + 어요 → 시 + 러요
싫어요 → [시러요] 'ㅎ' is dropped before vowel
젊 + 어요 → 절 + 머요
젊어요 → [절머요] to be young
💡 특별 케이스: 둘째 받침이 'ㅅ'이면 [ㅆ]로 강하게 발음돼요 (값이 → [갑씨], 없어요 → [업써요]).
When the SECOND consonant is ㅅ, it's pronounced as the tensed [ㅆ] when it links to the next syllable.
규칙 3-B: 겹받침 + 실질 형태소 (모음으로 시작) advanced
조사·어미가 아닌 실질적 의미가 있는 단어가 모음으로 시작하면, 연음하지 않고 받침 자체의 대표음(첫째 자음)을 옮겨 발음해요.
If the following word is a "real" word (noun/verb root) starting with a vowel — not a particle or ending — only the representative consonant moves over.
예시발음설명
값있어 (값 + 있다)[가비써]'있다'가 실질형태소 → 'ㅂ'(대표음)만 이동
없어요 (값 + 어요)[업써요]'-어요'는 어미 → 둘째 'ㅅ' 그대로 이동
This rule is more advanced and rare in everyday speech. Don't worry too much — focus on Rule 3-A first.

6. Summary — 한눈에 보는 요약

📌 겹받침 발음 3대 규칙

  1. 겹받침 ㄳ, ㄵ, ㄶ, ㄼ, ㄽ, ㄾ, ㅀ, ㅄ은 어말/자음 앞에서 각각 [ㄱ, ㄴ, ㄴ, ㄹ, ㄹ, ㄹ, ㄹ, ㅂ]으로 발음. (첫째 자음)
  2. 겹받침 ㄺ, ㄻ, ㄿ은 어말/자음 앞에서 각각 [ㄱ, ㅁ, ㅂ]으로 발음. (둘째 자음)
  3. 겹받침 + 모음 시작 조사/어미 → 둘째 받침이 다음 음절로 이동 ('ㅅ'은 [ㅆ]로 된소리).
