Large Units (큰 수)
⚠ Korean counts in 만(10,000) units — not thousands like English!
NumberKoreanRomanizationNote
1,000천cheon
10,000만man← unique Korean base unit (4자리마다 단위 변화)
100,000,000억eoknext major unit (≠ 영어 billion)
1,000,000,000,000조jonext unit after 억
영어: thousand → million → billion (3자리마다)
한국어: 천 → 만 → 억 → 조 (4자리마다) ← 중요!
예) 50,000 = 오만  ·  123,000 = 십이만 삼천

Common Mistakes by Foreigners
❌ 이십 살 (age 20)
이십 살 ✗ → 스무 살 ✔
Age under 100 → always Native Korean (살)
❌ 한 시 삼십 분 → shortcut
한 시 반 ✔
반(半) = 30 minutes — much more natural in speech!
❌ Using 개 for people
사람 두 개 ✗ → 두 명 ✔
개 = objects only / 명 = people / 마리 = animals
