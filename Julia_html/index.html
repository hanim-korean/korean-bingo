13. Notion / Netlify 사용법

EMBED
1. 새 폴더를 만들어요. 예: my-lesson
2. 그 안에 현재 HTML 파일을 index.html 로 저장해요.
3. 같은 폴더 안에 images 폴더를 만들어요.
4. 그 안에 사진 8장을 이렇게 넣어요:
bangchan.jpg, leeknow.jpg, changbin.jpg, hyunjin.jpg, han.jpg, felix.jpg, seungmin.jpg, in.jpg
5. 폴더 구조는 이렇게예요:
my-lesson / index.html / images / bangchan.jpg ...
6. Netlify에서 이 폴더 전체를 업로드해요.
7. 배포된 URL을 Notion에서 /embed로 넣어요.
사진 파일은 HTML과 같은 프로젝트 안에 넣어야 Netlify에서 잘 보여요. ㅑ, ㅕ, ㅛ, ㅠ는 읽기와 인지 중심으로 넣었기 때문에 아직 빠르지 않아요.
